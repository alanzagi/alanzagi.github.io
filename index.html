🚀 Projects

LANKAS - Aplikasi Pengelolaan Uang Kas

Lankas is a web-based class cash management system developed with Laravel. It allows efficient tracking of payments, expenditures, and financial reports. Features include profile management, CRUD functionality, and PDF generation.

MCatering - Company Profile & Ordering

Makmur Catering is a Laravel-based website designed for catering services, featuring a user-friendly interface for profile management, online ordering, and menu displays. Built with TailwindCSS and Flowbite, it offers an efficient platform for catering businesses.

LANBAKERY - Company Profile & Ordering

LANBAKERY is a complete bakery management website built with Laravel. It features product listings, order processing, and payment integration. Additionally, it includes an admin panel for the owner to manage stock and customize the website's appearance.
